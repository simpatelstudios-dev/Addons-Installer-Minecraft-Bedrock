## 📱 Tentang Aplikasi

**AIM** adalah aplikasi installer untuk addons dan map Minecraft yang mendukung berbagai format file dan versi Android.

Dirancang khusus untuk pengguna **Minecraft Bedrock Edition** di Android, AIM memudahkan proses instalasi addon tanpa perlu root atau aplikasi tambahan.

---

## ✨ Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| 📂 **Multi Format** | Support `.mcaddon` `.mcpack` `.mcworld` `.mctemplate` `.zip` |
| 🔍 **Auto Detect** | Otomatis deteksi tipe addon dari isi file |
| 🔄 **Konversi Otomatis** | Ubah `.zip` ke ekstensi Minecraft yang sesuai |
| 📲 **Direct Import** | Buka langsung di Minecraft (Android 11+) |
| 📁 **Manual Extract** | Ekstrak ke folder `com.mojang` (Android 10 ke bawah) |
| 🖥️ **UI Modern** | Desain gelap ala Minecraft dengan pixel art |

---

## 🔄 Dukungan Lintas Versi Android

```
┌─────────────────────────────────────────────┐
│              ALUR INSTALASI                  │
├─────────────────────────────────────────────┤
│                                             │
│  Pilih File (.zip / .mcpack / .mcaddon)     │
│              │                              │
│              ▼                              │
│      Deteksi Tipe Addon                     │
│              │                              │
│              ▼                              │
│    ┌─────────────────────┐                  │
│    │   Cek Android API   │                  │
│    └─────────────────────┘                  │
│         │           │                       │
│    API ≥ 30     API ≤ 29                    │
│         │           │                       │
│         ▼           ▼                       │
│   Direct Import   Extract ke               │
│   ke Minecraft    com.mojang/              │
│   via Intent      minecraftWorlds/         │
│                   behavior_packs/          │
│                   resource_packs/          │
└─────────────────────────────────────────────┘
```

- 🟢 **Android 11+ (API 30+)** → Import langsung menggunakan Intent ke Minecraft
- 🔵 **Android 10 ke bawah (API ≤ 29)** → Ekstrak otomatis ke folder `com.mojang.minecraftpe`

## 🚀 Cara Pakai

```
1. Install AIM di Android kamu
2. Buka AIM → izinkan akses storage
3. Tap "PILIH FILE ADDON"
4. Pilih file .zip / .mcpack / .mcaddon / .mcworld
5. Tap "INSTALL KE MINECRAFT"
6. Selesai! Buka Minecraft dan cek addon kamu 🎮
```
